## 🎮 How to Play

### 1️⃣ **Add Cards**
Tap **⚡ Quick** button → Select cards from grid → Done!

Or:
- Tap "✏️ Pick Cards" 
- Choose from visual grid
- Auto-adds +5 XP per card

### 2️⃣ **Get Analysis**
- Auto-analyzes when you have 2 hole + 3+ community
- See hand strength, win rate, outs, recommendation
- Gain **+25 XP** per analysis

### 3️⃣ **Unlock Achievements**
- Get your first flush → **♠️ Flush Master** (+75 XP)
- Analyze 10 hands → **📈 Getting Started** (+100 XP)
- Hit 95% win rate → **🚀 Monster Hand** (+75 XP)

### 4️⃣ **Level Up**
- Collect XP → Level up → Climb tiers
- Watch confetti fly! 🎉
- Unlock new tiers (Bronze → Diamond)

### 5️⃣ **Track Progress**
- 🏆 **Achievements** - See all unlocks
- 📊 **Stats** - View analytics
- ⏱ **History** - Browse past hands
